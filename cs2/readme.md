# 地图预定

# CS2 ZE指挥系统
- !vl 名字可以投票成为指挥  或者 !lv @me
- !glow @me 
- !defend 放一个防守图标
- !tracer 开关激光
- !beacon 光环
- !leaders 显示所有指挥
- !leaderhelp 显示帮助指令
- 可以给别人上效果 例如#userid 或者 名字 以及 @me
